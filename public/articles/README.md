# 記事アイキャッチ画像フォルダ

このフォルダに記事のアイキャッチ画像を配置してください。

## 使い方

1. このフォルダに画像ファイルを配置
2. 画像ファイル名と記事スラッグを指定
3. スクリプトで一括設定

## 推奨画像サイズ
- 横幅: 1200px以上
- アスペクト比: 16:9 または 4:3
- 形式: .jpg, .png, .webp

## 例
```
public/articles/sixtones.jpg → SixTONES関連記事
public/articles/bzai.jpg → B&ZAI関連記事
public/articles/acees.jpg → ACEes関連記事
```
